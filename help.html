FAQ

Here are some of the most frequently asked questions, hopefully you can find the answer to any question you have here.

  1. Something in a tutorial isn't working

    2. If something in a tutorial isn't working, please post a comment below with the issue that you are having. By posting a comment, anyone else who is having the same problem will be able to find the answer and others will be able to help you. If you have posted a comment and not recieved a reply, please contact me here: tutorials@dylanturner.tech.

  2. I found a typo

    3. If you find a typo, you can make a pull request on Github.

  3. I wan't to submit a tutorial

    4. If you would like to help by submitting a tutorial, you can make a pull request on Github.

  4. I have another problem

    5. If you have another problem, please contact me here: tutorials@dylanturner.tech. Please note that if you are struggling to understand a tutorial you can post in the comments section, please avoid spamming me with questions related to tutorials.



Community Guidelines

This website should be a safe environment for anyone to post and get help. Please avoid posting anything that could offend or insult someone else. Please just use common sense when posting. Anything that is deemed innapropiate will be removed.



Contact

To contact me you can email me here: tutorials@dylanturner.tech. I should reply within two days. If I haven't replied, try re-sending the email or posting a comment on one of the pages. If you believe there is anything missing from the website, have general feedback or wish to discuss something please don't hesitate to email me.

FAQ

Here are some of the most frequently asked questions, hopefully you can find the answer to any question you have here.

  1. The tutorial isn't working

    2. If something in a tutorial isn't working there are a few ways to go about it. Firstly, please post a comment below with the issue that you are having. By posting a comment, anyone else who is having the same problem will be able to find the answer and others will be able to help you. Another issue may be that you are using an outdated client or that the tutorial is outdated. Generally you will be able to see when the tutorial was posted, and any updates that have been made to that tutorial.

  2. I found a typo / mistake

    3. If you find a typo, please let me know by either posting a pull request on Github, posting a comment or contacting me.

  3. How do I submit a tutorial

    4. If you would like to help by submitting a tutorial, you can make a pull request on Github. After you have made the tutorial and it has been deemed working and appropiate it will be merged with the website. For any tutorial that you make, you will be given full credit for.

  4. Someone has posted an innapropiate comment

    5. If someone has posted something that is innapropiate in a comment that violates the community guidelines below, please contact me. You can also flag the comment, and if it is flagged over 3 times it will be temporarily removed untill an admin approves it.

  5. Who are you?

    6. My names Dylan, and you can find lots about me on my personal website here. You can also find information about this website from the about link in the menu.

  6. I have another problem

    7. If you have another problem, please contact me here: tutorials@dylanturner.tech. Please note that if you are struggling to understand a tutorial you can post in the comments section, please avoid spamming me with questions related to tutorials.



Community Guidelines

This website should be a safe environment for anyone to post and get help. This means that no one should insult anyone else because they post a "stupid" question or comment. Please avoid posting anything that could offend or insult someone else, use common sense when posting. Anything that is deemed innapropiate will be removed and you could be blocked from viewing or posting any comments on this website.



Contact

To contact me you can email me here: tutorials@dylanturner.tech. I should reply within two days. If I haven't replied, try re-sending the email or posting a comment on one of the pages. If you believe there is anything missing from the website, have general feedback or wish to discuss something please don't hesitate to email me. Below are the rules in more detail.



Keep It Civil

We’re going to get into the thick of a lot of heated discussions and that’s okay. These discussions often entail topics that we all personally care a lot about and will passionately defend. But in order for discussions to thrive here, we need to remember to criticize ideas, not people.

So, remember to avoid:

  • name-calling
  • personal attacks
  • Responding to a post’s tone instead of its actual content.

Comments that we find to be hateful, inflammatory, or harassing may be removed. If you don’t have something nice to say about another user, don't say it. Treat others the way you’d like to be treated.

I would add more to this, but I'm sure that you get the gist.
